História do Linux

O Linux surgiu em 1991, criado por Linus Torvalds, um estudante finlandês de ciência da computação. Inspirado pelo sistema UNIX, ele desenvolveu um novo núcleo de sistema operacional — o kernel Linux — como um projeto pessoal, com o objetivo de criar uma alternativa livre e acessível. Desde então, o Linux evoluiu rapidamente graças à colaboração da comunidade global de desenvolvedores. Com base na filosofia do software livre e de código aberto, qualquer pessoa pode estudar, modificar e distribuir o sistema, o que impulsionou sua adoção em larga escala. Hoje, o Linux está presente em servidores, supercomputadores, dispositivos móveis, roteadores, sistemas embarcados e até em plataformas espaciais. Sua estabilidade, segurança e flexibilidade fazem dele um dos sistemas operacionais mais influentes do mundo.

Recursos e ferramentas

O terminal é uma ferramenta poderosa para controle total do sistema.

Ferramentas como apt, dnf e pacman facilitam a instalação de softwares.

GNOME, KDE e XFCE são opções populares para interfaces visuais.

O Linux permite criar e gerenciar múltiplos usuários e grupos, além de configurar permissões com comandos como chmod e chown.

O Linux é amplamente utilizado em servidores devido à sua estabilidade e segurança. Ele suporta servidores web (Apache, Nginx), bancos de dados (MySQL, PostgreSQL), servidores de arquivos (Samba, NFS) e muito mais, tornando-se a base da infraestrutura da internet.

Comandos no Terminal

Comandos

ls

Mostra o que tem dentro do diretório que você está

cd

Troca de diretório, por exemplo, cd assets te joga para o diretório assets

touch

Cria um arquivo novo ou "toca" num já existente para mudar sua data de alteração

rm

Remove um arquivo, com modificador -r pode remover uma pasta

Atençao! Nunca rode o comando rm -rf / isso remove todo seu sistema!

mkdir

Cria um diretório novo dentro do diretório que você está

rmdir

Remove um diretório, porém ele deve estar vazio, se houver algo dentro dele você deve usar rm -r diretório

whoami

Mostra em qual usuário do sistema você está logado

pwd

Mostra o caminho absoluto (completo) do diretório que você está no momento

cp

Copia um arquivo ou pasta para outro diretório

mv

Move um arquivo ou pasta para outro diretório

ps

Lista os processos que estão executando no sistema operacional

kill

Cessa um processo por meio de seu número de identificação (PID)

ping

Testa a conexão
